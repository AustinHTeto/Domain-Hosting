Domain dan Hosting

Penjelasan, Pengertian dan fungsi

1. Apa Itu Domain?

Domain adalah alamat unik yang digunakan untuk mengakses sebuah website di internet. Tanpa domain, kita harus mengingat alamat IP (seperti 192.168.1.1)

Fungsi Domain:

  • Menjadi alamat unik untuk mengakses website.
  • Mempermudah pengunjung mengingat situs web kamu.
  • Mencerminkan identitas atau brand bisnis.
Contoh:

Alamat website www.tokopedia.com memiliki domain “tokopedia.com”.

2. Apa Itu Hosting?

Hosting adalah tempat untuk menyimpan semua data dan file website, seperti gambar, teks, video, dan database, agar bisa diakses oleh pengunjung melalui internet.

Fungsi Hosting:

  • Menyimpan file dan data website.
  • Menjaga agar website bisa diakses 24 jam.
  • Mendukung kecepatan dan keamanan situs.
Contoh:

File gambar, artikel, dan desain aplikasi www.tokopedia.com disimpan di server hosting milik Tokopedia atau penyedia hosting profesional.

3. Apa Itu Domain Extension?

Domain extension adalah huruf huruf akhiran pada nama domain yang menunjukkan kategori atau asal domain tersebut.

Contoh Ekstensi Domain:
  • .com → komersial atau bisnis umum
  • .org → organisasi non-profit
  • .edu → lembaga pendidikan
  • .id → domain Indonesia
  • .tv → domain Tuvalu
